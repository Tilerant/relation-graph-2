# 🚀 图谱笔记系统启动指南

## 快速启动

### 方法一：自动启动（推荐）
1. **双击运行**: `start_system.bat`
2. 系统会自动启动后端和前端服务

### 方法二：手动启动

#### 1. 启动后端
```bash
cd backend
python main.py
```

**说明**: 
- 系统会自动寻找可用端口（3001-3020）
- 如果看到"INFO: Uvicorn running on http://127.0.0.1:XXXX"表示启动成功
- 端口号可能不是3001，前端会自动检测

#### 2. 启动前端
```bash
cd frontend  
npm run dev
```

## ❗ 常见问题解决

### 问题1: Windows 端口权限错误
```
ERROR: [Errno 13] error while attempting to bind on address
```

**解决方案**:
1. **以管理员身份运行**（推荐）
   - 右键点击"命令提示符" → "以管理员身份运行"
   - 或双击 `backend/run_as_admin.py`

2. **检查端口占用**
   ```bash
   netstat -ano | findstr :3001
   ```

3. **关闭防火墙/杀毒软件**（临时）

### 问题2: 后端启动无反应
- 检查Python版本（需要3.8+）
- 安装依赖：`pip install -r requirements.txt`
- 查看是否有错误信息

### 问题3: 前端无法连接后端
- 前端会自动检测后端端口（3001-3020, 8000-8001）
- 查看浏览器控制台的连接日志
- 确保后端已成功启动并显示端口信息

## 📊 验证系统运行

### 后端验证
访问：http://localhost:XXXX （XXXX为显示的端口号）
应该看到：`{"message": "图谱系统 API v2.0", "status": "running"}`

### 前端验证  
访问：http://localhost:5173
- 应该看到图谱系统界面
- 左侧边栏显示"创建知识库"按钮
- 控制台显示后端连接成功

## 🧪 测试功能

1. **创建知识库**：点击左侧"创建知识库"按钮
2. **查看示例数据**：系统会自动加载示例知识库
3. **编辑节点**：双击节点打开右侧编辑面板
4. **拖拽节点**：在白板中拖动节点位置

## 📞 技术支持

如果遇到其他问题：
1. 查看控制台错误信息
2. 检查端口是否被占用
3. 尝试重启系统
4. 临时关闭杀毒软件

---
💡 **提示**: 首次启动建议使用管理员权限，避免权限相关问题。
